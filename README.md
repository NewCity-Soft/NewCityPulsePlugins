# NewCityPulsePlugins

The official plugins store of **New City Pulse** / **新城脉动** 的官方插件商店仓库。

## 简介 (Introduction)

本仓库是《新城脉动》官方维护的云端插件源。你可以通过游戏内的插件管理器直接浏览、下载并安装这些插件，以扩展游戏功能。

This repository is the official cloud plugin source for *New City Pulse*. You can browse, download, and install plugins directly through the in-game plugin manager to extend your game experience.

## 如何提交插件 (How to Contribute)

1. **Fork** 本仓库。
2. 在plugin目录下创建一个新文件夹，包含你的插件代码和 `manifest.json`。
3. 在根目录的 `registry.json` 中添加你的插件信息。
4. 提交 **Pull Request**。

## 插件源配置 (Plugin Source Configuration)

游戏默认从以下地址获取插件列表：
`https://raw.githubusercontent.com/NewCity-Soft/NewCityPulsePlugins/main/registry.json`
